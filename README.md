### 프로젝트 상세 소개

💡 **핵심 기능**

1. 'scroll' 이벤트를 사용하여 페이지 스크롤 할때 네비의 높이와 마진값이 변경이 되도록 하였습니다
2.  메뉴 항목에 마우스를 올리면 서브메뉴를 표시하고, 마우스가 해당영역에서 없어지면 서브메뉴를 숨기도록 하였습니다.
3.  배너에 Swiper 슬라이더 추가 및 자동재생 버튼, 게이지바를 추가하였습니다.
